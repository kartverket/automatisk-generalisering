class input_data.input_datasets.DatasetNamespace(data: dict)[source]

Bases: object

Lightweight container class that exposes dataset feature classes as attributes.

The class dynamically assigns each key/value pair from the input dictionary as an attribute on the instance. Keys become attribute names, and values (f.ex. Path objects or strings) become attribute values.

<dynamic>

Each key in the provided dictionary becomes an attribute on the instance, pointing to the corresponding dataset path.
